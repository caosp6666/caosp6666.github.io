Linux zsh

Posted on 2020-08-10

先查看自己本机的shell都有哪些

cat /etc/shells

安装zsh

(Ubuntu)sudo apt-get install zsh

然后再查看shell

cat /etc/shells

这时会发现多了两个

安装oh my zsh

wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh（需要翻墙，如果不翻墙就去gitee下载源码）

安装成功后会

查看当前使用的shell

echo $SHELL

此时还是/bin/bash，使用chsh -s /bin/zsh，输入密码，切换到zsh，重启一个Terminal，即可切换到zsh

打开.zshrc配置文件，修改主题为muse
