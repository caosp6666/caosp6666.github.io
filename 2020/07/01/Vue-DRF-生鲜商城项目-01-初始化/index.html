Vue+DRF 生鲜商城项目-01-初始化

环境搭建

windows(cmd中)：

创建虚拟环境

C:\Users\leishen>mkvirtualenv foodshop

进入虚拟环境

workon foodshop

查看已安装包

pip list

安装django==2.2.6

pip install django==2.2.6

安装restframework

pip install djangorestframework

安装markdown

pip install markdown

安装django-filter

pip install django-filter

安装pillow（图片处理）

pip install pillow

通过pycharm新建项目

配置linux的数据库

设置settings.py中的数据库部分

此时运行，会发现报错如下

因此安装mysqlclient，然后再运行正常

整理项目架构

新建apps目录，用于存放应用

新建extra_apps目录，用于存放第三方的包，可以修改其中的源码，而不是直接安装使用

新建media目录，用于存放图片

新建db_tools目录，用于存放数据库初始化的脚本

在settings中，将apps和extra_apps目录加入到项目搜索目录

这样的目的是可以直接from users import xxx，而不是from apps.users import xxx

前端页面与apps分离设计

将前端文件解压，进入文件夹cmd，执行cnpm install下载需要的包

运行cnpm run dev然后即可通过localhost:8000查看前端页面

根据前端可将apps分为user / trade / goods / user_operations四个部分。

通过pycharm的Run manage.py Task 创建app

将建好的所有app放到apps文件夹下

全局私密信息分离

创建一个文件命名为secret_key.py，在settings.py中import * 然后把写死的变量分离出来。

将secret_key.py添加到gitignore
