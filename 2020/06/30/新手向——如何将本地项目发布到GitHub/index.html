新手向——如何将本地项目发布到GitHub

Posted on 2020-06-30 Edited on 2020-07-10

具体操作

首先在GitHub建立一个new repository

新建一个文件夹，在文件夹中右键git bash here

初始化git init

将已有的项目文件复制到这里，然后查看git status

将所有文件添加到git仓库git add .

进行提交git commit -m "first commit"

（插曲：配置Github的SSH登陆）

将GitHub仓库与本地仓库关联：

git remote add origin https://github.com/caosp6666/Dream-OnlineCourse.git

你新创建的那个仓库里面的README文件不在本地仓库目录中

通过git pull --rebase origin master进行合并

再将项目推到远端git push -u origin master （后面的改动不需要-u）
